root scope is the ultimate parent scope within each application. as code it can be writted "var rootScope = angular.element(document).scope()". scope is inherited from parent to child elements (like prototypical inheritance) These create their own scope: ng-controller, ng-repeat, ng-if (actually destroys the scope every time it is false and creates a new one every time it is true - be careful with this one!), ng-view, ng-switch, ng-include This means that one must first make an object in the parent scope and assign the value to a property within that object.
